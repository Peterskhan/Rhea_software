Macro definitions.

Author
Peter Gyulai
Version
1.0.2
Date
2020.03.17

This header file contains all the macro definitions used by the BME280 driver library. These macros are used throughout the implementation and accessible by the end user. The general convention is that all macros are preceeded by the BME280_ prefix.
