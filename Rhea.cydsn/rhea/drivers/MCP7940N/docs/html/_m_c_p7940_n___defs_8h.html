Detailed Description

Macro definitions.

Author
Peter Gyulai
Version
1.0.0
Date
2020.02.26

This header file contains all the macro definitions used by the MCP7940N driver library. These macros are used throughout the implementation and accessible by the end user. The general convention is that all macros are preceeded by the MCP7940N_ prefix.

Macro Definition Documentation

◆ MCP7940N_MFP_ALARM

#define MCP7940N_MFP_ALARM   (uint8_t)(0xBF)

Defines a bitmask for clearing the SQWEN bit, resulting in alarm output mode. If none of the alarms are enabled, the operation results in GPIO mode instead. Use bitwise AND (&) on the CONTROL register.

Example:

MCP7940N_GetLocalRegisters()->CONTROL &= MCP7940N_MFP_ALARM;

◆ MCP7940N_MFP_GPIO

#define MCP7940N_MFP_GPIO   (uint8_t)(0x8F)

Defines a bitmask for clearing SQWEN and ALMxEN bits, resulting in GPIO output mode. Use bitwise AND (&) on the CONTROL register.

Example:

MCP7940N_GetLocalRegisters()->CONTROL &= MCP7940N_MFP_GPIO;

◆ MCP7940N_MFP_SQUARE

#define MCP7940N_MFP_SQUARE   (uint8_t)(0x40)

Defines a bitmask for setting the SQWEN bit, resulting in squarewave output mode. Use bitwise OR (|) on the CONTROL register.

Example:

MCP7940N_GetLocalRegisters()->CONTROL |= MCP7940N_MFP_SQUARE;
