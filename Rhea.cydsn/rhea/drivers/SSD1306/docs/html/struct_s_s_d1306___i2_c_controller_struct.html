Detailed Description

Structure for passing the I2C bus controller functions, that are used by the SSD1306 driver.

The function pointer members must be provided for the library to control trafic on the I2C bus. The library is independent from the specific I2C implementation and only needs these callbacks to operate. If the I2C implementation does not provide functions with these specific function signatures, consider creating proxy functions to pass.
